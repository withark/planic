## 관리자 확인 화면 (프로덕션에서 시각적으로 확인된 화면만)

코드에 `page.tsx`가 있거나 사이드바에 있다는 이유만으로 포함하지 않는다. **운영 프로덕션에서 실제로 열림·확인된** 관리자 검수 표면만 아래에 둔다.

| 구분 | 경로 | 용도(요약) |
|------|------|------------|
| 운영 개요 | `/admin` | 대시보드 |
| 문서 생성 운영 | `/admin/generation-logs` | 생성 로그 |
| 비즈니스 운영 | `/admin/users` | 사용자 관리 |
| | `/admin/subscriptions` | 구독 현황 |
| | `/admin/payments` | 결제 관리 |
| 시스템·로그 | `/admin/logs` | 에러 로그 |

**이 목록에서 제외한 것(코드상 존재하나 프로덕션 시각 확인 전제 없음)**  
`/admin/samples`, `/admin/engines`, `/admin/system` 등 — 프로덕션에서 별도로 확인되기 전까지 최종 클로저 체크리스트에 넣지 않는다.

**참고**: 동적 상세만 존재하는 **`/admin/quotes/[id]`** 등은 필요 시 개별 건 추적용으로만 쓰고, 목록 URL인 `/admin/quotes`는 라우트가 없으므로 “목록 화면”으로 쓰지 않는다.
